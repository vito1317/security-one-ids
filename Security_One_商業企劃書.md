## 1. 執行摘要

### 1.1 產品概述

**Security One** 是一套完整的企業級網路安全防護解決方案，整合了 **Web 應用程式防火牆（WAF）** 與 **入侵偵測/防禦系統（IDS/IPS）**，為企業提供從邊界防護到內部偵測的全方位安全保障。

### 1.2 核心價值主張

| 價值 | 說明 |
|-----|------|
| 🛡️ **全方位防護** | 整合 WAF 與 IDS/IPS，打造縱深防禦架構 |
| 🤖 **AI 智能驅動** | 運用本地 LLM 進行智能威脅分析 |
| 📊 **視覺化監控** | 專業 SOC 儀表板與 3D 攻擊視覺化 |
| 🚀 **快速部署** | Docker 容器化，15 分鐘內完成部署 |
| 💰 **成本效益** | 以訂閱模式提供企業級安全能力 |

### 1.3 市場機會

- 全球網路安全市場預計 2027 年達到 **3,760 億美元**
- 亞太區 WAF 市場年複合成長率達 **17.8%**
- 勒索軟體攻擊成本預計 2031 年達到 **2,650 億美元**

## 4. 產品方案

### 4.1 Security One WAF

#### 核心功能模組

| 模組 | 功能說明 |
|-----|---------|
| **多站點管理** | 單一控制台管理無限網域 |
| **反向代理** | OpenResty 高效能代理引擎 |
| **負載均衡** | Round Robin / IP Hash / 最少連線 |
| **SSL 管理** | Let's Encrypt 一鍵簽發 |

#### 攻擊防護引擎

| 攻擊類型 | 防護能力 |
|---------|---------|
| **SQL Injection** | ✅ 多層偵測、編碼繞過防禦 |
| **XSS** | ✅ DOM/Stored/Reflected 全覆蓋 |
| **RCE** | ✅ 命令注入、反向 Shell 防禦 |
| **Path Traversal** | ✅ 目錄遍歷、敏感檔案保護 |
| **OGNL / Java 反序列化** | ✅ Struts2、WebLogic 漏洞防護 |
| **掃描器偵測** | ✅ 自動化工具識別與封鎖 |

#### AI 智能防禦

- 整合本地 **Ollama LLM** 引擎
- 變形攻擊語意分析
- 自動規則建議與生成
- CVE 漏洞追蹤與規則派發

#### 進階功能

| 功能 | 說明 |
|-----|------|
| **CC/DDoS 防護** | 速率限制、CAPTCHA 驗證 |
| **Passkey 認證** | WebAuthn 無密碼登入 |
| **站點認證** | WAF 層級的訪問控制 |
| **自動封鎖規則** | 行為驅動的智能封鎖 |

### 4.2 Security One IDS/IPS

#### 分散式架構

```
┌─────────────────────────────────────────────────────────┐
│                    WAF Hub (中央控制)                    │
│  ┌─────────────────────────────────────────────────────┐ │
│  │  Agent 管理 │ 規則派發 │ 告警整合 │ SOC Dashboard │ │
│  └─────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────┘
                           │
        ┌──────────────────┼──────────────────┐
        ▼                  ▼                  ▼
   ┌─────────┐        ┌─────────┐        ┌─────────┐
   │ IDS Agent │      │ IDS Agent │      │ IDS Agent │
   │ Server A  │      │ Server B  │      │ Server C  │
   └─────────┘        └─────────┘        └─────────┘
```

#### 四大偵測引擎

| 引擎 | 偵測方式 | 適用場景 |
|-----|---------|---------|
| **特徵引擎** | 正規表達式匹配 | 已知攻擊特徵 |
| **異常引擎** | 基線偏離偵測 | 異常行為識別 |
| **行為引擎** | 行為模式分析 | 掃描/爬蟲/暴力破解 |
| **AI 引擎** | LLM 智能分析 | 變形攻擊偵測 |

#### IDS Agent 功能

| 功能 | 說明 |
|-----|------|
| **日誌監控** | 自動探索 Nginx/Apache 日誌 |
| **即時偵測** | 即時分析並告警 |
| **自動封鎖** | iptables / WAF 雙層封鎖 |
| **流量分析** | QPS、IP 頻率、URL 模式 |
| **Hub 同步** | 規則同步、告警上報 |
| **多通道通知** | Telegram / Email / Webhook |

- **AI 威脅報告**：自動生成安全態勢報告
- **系統監控**：CPU / RAM / 磁碟 / 網路即時狀態

#### 公開分享功能
